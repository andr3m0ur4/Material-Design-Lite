CARTÕES

Um cartão é uma área para exibir um conteúdo, como texto e imagens que geralmente promove algum tipo de ação. Por exemplo, uma imagem com um botão "like" ou "compartilhar", uma lista de músicas com o botão "tocar", ou talvez um nota com um botão "Salvar".

O componente cartão do Material Design Lite (MDL) é um elemento da interface do usuário que representa um pedaço de papel virtual que contém dados relacionados - como uma foto, algum texto e um link - que são todos sobre um único assunto.

Os cartões são um meio conveniente de exibir coerentemente conteúdo relacionado que é composto de diferentes tipos de objetos. Eles também são adequados para apresentar objetos semelhantes cujo tamanho ou ações suportados podem variar consideravelmente, como fotos com legendas de comprimento variável. Os cartões têm uma largura constante e uma altura variável, dependendo do conteúdo.
